자바에서 Jackson이라는 강력한 JSON 처리 라이브러리를 이용해 간단하게 직렬화와 역직렬화를 할 수 있다.

#### 직렬화란?
Java 객체를 JSON 문자열로 변환(직렬화)하는 것을 의미한다.

#### 역직렬화란?
JSON 문자열을 Java 객체로 변환(역직렬화)하는 것을 의미한다.
